# Collapse 折叠面板
通过折叠面板收纳内容区域。
## 基础用法
每次能展开多个面板。
<div>
    <div>
        <tass-collapse :posts="list" :types="true">
           <div style="height: 200px"></div>
        </tass-collapse>
    </div>
</div>
