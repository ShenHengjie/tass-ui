# DatePicker 日期选择器

用于选择或输入日期

## 选择某一天

以”日“为基本单位，基础的日期选择控件

<div class=example>
  <Base/>
</div>
<details>
<summary>展开查看</summary>

```vue
<template>
  <tass-date-picker v-model="nowDate"></tass-date-picker>
</template>
<script setup lang="ts">
  import { ref } from 'vue';
  const nowDate = ref(new Date());
</script>
```

</details>
